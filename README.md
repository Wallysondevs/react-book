# ⚛️ React — Guia Completo em Português

  Guia interativo de **React** em PT-BR — 84 capítulos, exemplos prontos, padrões modernos (2026).

  🌐 **Online:** https://wallysondevs.github.io/react-book/

  ## Conteúdo

  - **Boas-vindas** — História, filosofia, React vs Vue
  - **Setup** — Node, Vite, estrutura de projeto, primeiro componente
  - **JSX & Componentes** — JSX, funcionais, props, children, fragments, render condicional
  - **Listas & Keys** — boas práticas, key como reset
  - **Hooks Essenciais** — useState, useEffect, useRef, useContext, useReducer, useMemo, useCallback
  - **Hooks Avançados** — Custom hooks, useLayoutEffect, useTransition, useDeferredValue, useId
  - **Componentes Avançados** — forwardRef, memo, Portals, Error Boundaries, Suspense
  - **Eventos & Forms** — eventos, controlado vs uncontrolled, validação Zod, react-hook-form
  - **Estilização** — CSS Modules, styled-components, Tailwind v4, Emotion
  - **Roteamento** — React Router, nested, protected, lazy
  - **Estado Global** — Context, Redux Toolkit, Zustand, Jotai, Recoil
  - **Data Fetching** — fetch, Axios, React Query, SWR, Suspense Data
  - **Server & SSR** — CSR vs SSR, Next.js, Server Components, Streaming
  - **Performance** — Profiler, code-splitting, virtualização, Web Vitals, React Compiler
  - **Testes** — Vitest, Testing Library, MSW, Playwright
  - **TypeScript** — Setup, props, hooks, generics
  - **Build & Deploy** — Vite, Vercel, Netlify, GitHub Pages
  - **Padrões** — Composição, render props, HOCs
  - **Extras** — i18n, acessibilidade, referências

  ## Stack

  - ⚛️ React 18 + TypeScript
  - ⚡ Vite 5
  - 🎨 Tailwind CSS v4 + shadcn/ui
  - 🧭 wouter (hash routing) — funciona em GitHub Pages sem configuração extra
  - 🔄 TanStack Query

  ## Rodar localmente

  ```bash
  pnpm install
  pnpm dev
  ```

  ## Deploy

  GitHub Actions builda e publica em GitHub Pages a cada push na `main`. Veja `.github/workflows/deploy.yml`.
